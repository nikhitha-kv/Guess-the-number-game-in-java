# Guess the Number Game

Welcome to the Guess the Number game! This is a simple Java program where the computer generates a random number, and the user tries to guess it within a specified range.

## How to Play

1. **Generate a Random Number**: The computer generates a random number within a specified range, such as 1 to 100.

2. **User Input**: The user is prompted to enter their guess for the generated number.

3. **Comparison**: The program compares the user's guess with the generated number and provides feedback on whether the guess is correct, too high, or too low.

4. **Repeat**: Steps 2 and 3 are repeated until the user guesses the correct number.

## Additional Features

- **Limited Attempts**: The number of attempts the user has to guess the number is limited.
- **Multiple Rounds**: The user can play multiple rounds of the game.
- **Score Display**: The user's score is displayed, based on the number of attempts taken or rounds won.

## Running the Program

To run the Guess the Number game, follow these steps:

1. Clone the repository to your local machine:

    ```
    git clone https://github.com/nikhitha-kv/Guess-the-number-game-in-java.git
    ```

2. Navigate to the project directory:

    ```
    cd guess-the-number-in-java
    ```

3. Compile the Java files:

    ```
    javac GuessTheNumberUI.java
    ```

4. Run the program:

    ```
    java GuessTheNumberUI
    ```

5. Follow the on-screen instructions to play the game!
